# Wonder

Mobile itinerary planner MVP built with Expo React Native.

## What this app does

- Creates **individual accounts** with a quick onboarding survey.
- Stores each profile's:
  - interests
  - food preferences
  - availability preferences
  - default drive radius
  - home city and budget tier
- Supports three planning modes:
  - **Solo**
  - **Friendship**
  - **Date**
- Builds **3 itinerary options** at a time based on:
  - selected participants
  - budget per person
  - city
  - drive radius
  - time window (day type + time slot)
  - food/activity toggles
- Includes event types such as hikes, concerts, museums, food spots, sports, and tours.

## Run locally

```bash
cd wonder-mobile
npm install
npm run start
```

Then launch Android/iOS/web from the Expo CLI prompt.
